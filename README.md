# EmprestAi Spring API

> ℹ️ **Aviso:** Este projeto utiliza nomes de classes, métodos e variáveis em português com o objetivo de facilitar o aprendizado e a compreensão do domínio de negócio. A estrutura do projeto segue padrões amplamente utilizados no mercado, permitindo fácil adaptação para inglês, se necessário.
> 
O EmprestAi é uma API REST desenvolvida em Java com Spring Boot, responsável por realizar simulações de empréstimos, aplicando regras de negócio, cálculos financeiros e validações de dados.

O projeto foi criado com foco em boas práticas de desenvolvimento backend e arquitetura em camadas, sendo projetado para integração com uma aplicação frontend desenvolvida em Angular.
